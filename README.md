# e-plantShopping 🌱

## Nom du dépôt
e-plantShopping

## Description du projet
L'application **e-plantShopping** est une plateforme de commerce électronique dédiée à l'achat et à la vente de plantes d'intérieur et d'extérieur. Cette application permet aux utilisateurs de parcourir un catalogue de plantes, d'ajouter des articles à leur panier et de gérer leurs commandes.

## Fonctionnalités
- Catalogue de plantes avec filtres par catégorie
- Panier d'achat dynamique
- Gestion de l'état avec Redux
- Interface utilisateur responsive avec React

## Technologies utilisées
- React.js
- Redux / Redux Toolkit
- JavaScript (ES6+)
- CSS / Bootstrap

## Installation

```bash
# Cloner le dépôt
git clone https://github.com/kynn237/e-plantShopping.git

# Accéder au dossier du projet
cd e-plantShopping

# Installer les dépendances
npm install

# Lancer l'application
npm start
